## Naming Convention

Per course guidelines (see [pilot 009 template](../../../../../009-Course-Repository-Template-and-Guidelines/README.md)):

- **Week/topic pattern (preferred):** `wkNN_<topic>_<index>.png`
  - e.g., `wk08_pickle_rick_nmap_1.png`
  - e.g., `wk13_boiler_sar2html_rce_3.png`
- **Generic pattern:** `Screenshot<index>_<short-desc>.png`
- All lowercase with dashes/underscores; no spaces
