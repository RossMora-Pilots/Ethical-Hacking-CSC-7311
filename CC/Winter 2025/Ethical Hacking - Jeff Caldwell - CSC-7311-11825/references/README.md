# References

Study companions and reference materials consolidating course content for later reuse. These documents go beyond lecture notes — they synthesize the practical lab work with industry frameworks and Canadian legal context to serve as a working reference for offensive security engagements.

**Suggested reading order for new readers:**

1. **[methodology.md](methodology.md)** — Start here. The six-phase pentest lifecycle provides the mental model that every other document builds on. *(~5 min read)*
2. **[cyber-kill-chain.md](cyber-kill-chain.md)** — Understand the attacker's strategic progression and the defensive controls that break it. *(~4 min read)*
3. **[legal-and-ethics.md](legal-and-ethics.md)** — The non-negotiable foundation: Canadian Criminal Code, PIPEDA, authorization requirements, and the "Is this legal?" decision tree. *(~6 min read)*
4. **[tools.md](tools.md)** — Practical reference for every tool used in the course, with canonical commands and a decision tree for tool selection. *(~8 min read, or use as reference)*
5. **[owasp-top-10.md](owasp-top-10.md)** — The 2021 OWASP Top 10 mapped to the course's CTF labs, with examples and mitigations. *(~7 min read)*
