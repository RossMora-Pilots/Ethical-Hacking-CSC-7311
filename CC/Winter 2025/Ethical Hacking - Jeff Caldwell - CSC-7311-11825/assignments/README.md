## Why the markdown re-renderings?

The original DOCX submissions were screenshot-heavy documents optimized for graded submission on Moodle. The portfolio markdown versions:

- Preserve the full tool → reason → expected → actual → screenshot narrative
- Add methodology mapping (Cyber Kill Chain, MITRE ATT&CK, OWASP)
- Add cross-references to other weeks and reference documents
- Render well on GitHub and in static site generators
- Remain readable 5 years from now when DOCX binaries may be inaccessible
